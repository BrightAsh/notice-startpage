주요 업데이트 내용
기본 대화 모델이 GPT-5.3 Instant로 업데이트
2026-03-03
설명
  • OpenAI는 GPT-5.3 Instant를 ChatGPT에서 가장 많이 쓰이는 기본 대화 모델의 업데이트로 소개했습니다.
  • 더 정확한 답변과 웹 검색 시 더 풍부하고 맥락 있는 결과를 강조했습니다.
  • 불필요한 단서와 경고 표현을 줄이고 더 자연스럽고 간결한 대화 스타일을 지향합니다.
  • ChatGPT 전체 사용자와 API의 gpt-5.3-chat-latest로 바로 제공되어 별도 설정 없이 일상 대화 품질이 직접 좋아지는 변화입니다.
