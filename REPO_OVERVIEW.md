# Repository Overview

## 목적
이 저장소는 GitHub Pages로 배포되는 사내 AI SaaS 시작페이지입니다. 사용자 화면(`index.html`)과 관리자 화면(`admin.html` + `admin.js`)으로 구성되어 있습니다.

## 주요 구성
- `index.html`: 서비스 카드, 공지사항, AI Update News, 가이드 PDF 패널을 렌더링합니다.
- `content.json`: 서비스 목록, 공지, 뉴스 메타데이터를 저장하는 단일 데이터 소스입니다.
- `admin.html`, `admin.js`: GitHub PAT로 로그인 후 `content.json`과 `files/*.pdf`를 수정/업로드하는 관리자 기능입니다.
- `files/*.pdf`: 서비스별 가이드북 PDF입니다.
- `News/*.html`: 뉴스 상세 콘텐츠 HTML입니다.

## 동작 흐름 요약
1. 사용자 페이지는 로드 시 `content.json`을 읽어 서비스/공지/뉴스를 화면에 반영합니다.
2. 서비스 이름 기준으로 `files/{서비스명}.pdf` 가이드 링크를 생성합니다.
3. 관리자 페이지는 GitHub Contents API를 통해 JSON/PDF를 저장하고, 변경사항을 원격 저장소에 반영합니다.

## 현재 데이터 상태(샘플)
- 서비스 목록: AI 도구들이 다수 등록되어 있고, 일부 테스트성 데이터(예: "구글", "Test 수정")가 포함되어 있습니다.
- 공지사항: 구독 변경 알림과 테스트성 항목이 함께 존재합니다.
- 뉴스: 다수 항목이 등록되어 있으며 일부는 동일 상세 파일을 재사용합니다.

## 빠른 확인 포인트
- URL/도메인/서비스명 불일치 여부
- 테스트용 문구 제거 필요 여부
- 뉴스 상세 파일 매핑 중복 여부
